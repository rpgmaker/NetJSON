How to Use
==========

```csharp
var myObject = new SimpleObject(){ ID = 100, Name = "Test", Value = "Value" };

var json = NetJSON.Serialize(myObject);

var recreatedObject = NetJSON.Deserialize<SimpleObject>(json);
```
